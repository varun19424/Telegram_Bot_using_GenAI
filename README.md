# Telegram_Bot_using_GenAI

# How to run?

### Steps 1:

clone the repository

```bash
git clone https://github.com/
```

### Steps 2:

Create a virtual environment

```bash
conda create -n telebot python=3.9 -y
```

```bash
conda activate telebot
```

```bash
pip install -r requirements.txt
```
